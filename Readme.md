# Movies Api project

movies api project using `MERN stack`

### important comments

1. `npm init -y`
2. `npm i`
3. `git init`
4. `git add .`
5. `git commit -m "first commit"`
6. `git push -u origin main or git push`
7. `npm i dotenv`
8. `npm install mongoose`
9. `npm i nodemon`

### Database options

1. `SQL - (MySQL, PostgreSQL, MariaDB)`
2. `No SQL - MongoDB`
